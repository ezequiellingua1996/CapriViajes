# Capri Viajes

Este proyecto incluye dos páginas estáticas para la marca Capri Viajes: una versión pública para mostrar el sitio y una versión de administración para gestionar el contenido.

## Archivos incluidos

- [index.html](index.html): sitio web público con secciones de inicio, destinos, paquetes, grupales, testimonios, quiénes somos y llamadas a WhatsApp.
- [admin.html](admin.html): panel administrativo para editar información del sitio, gestionar contenido y ajustar credenciales desde la sección de Seguridad.

## Funcionalidades principales

### Sitio público
- Diseño moderno y responsive.
- Sección principal con hero, destacados y bloques informativos.
- Páginas y vistas para paquetes, grupales y detalle de viajes.
- Integración visual con WhatsApp y navegación del sitio.

### Panel de administración
- Login para ingresar al panel.
- Secciones para manejar paquetes, grupales, testimonios, destinos, contacto y quiénes somos.
- Guardado de datos en el navegador mediante Local Storage.
- Opción para cambiar usuario y contraseña desde Seguridad.

## Cómo usar

1. Abrí [index.html](index.html) en el navegador para ver la web pública.
2. Abrí [admin.html](admin.html) para entrar al panel administrativo.
3. Si no se modificaron las credenciales, el acceso por defecto suele ser:
   - Usuario: `admin`
   - Contraseña: `capri2025`

## Notas técnicas

- El proyecto está desarrollado con HTML, CSS y JavaScript sin dependencias externas.
- No requiere instalación ni compilación.
- Para una mejor experiencia, se recomienda abrir los archivos desde un navegador moderno.
